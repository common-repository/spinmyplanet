Spinmyplanet is a plugin that helps you rotate images beautifully.
Spinmyplanet has a woocommerce product spin feature which requires woocommerce plugin to be installed for this plugin to work for product spin beautifully.


== Instructions ==

Features: For images spin

1. Drag and drop re-ordering
2. Add images in Images tab in admin panel
3. Add the shortcode generated to any post of your choice
4. If more than one image is added to the gallery, the images become grouped
5. Add multiple images to the gallery at once


Features: For woocomerce product image spin

1. Add product gallery images
2. If more than one image is added to the gallery, the product gets a drag to spin feature
3. Add multiple images to the gallery at once


== Installation ==
1. Upload the entire 'Spinmyplanet' folder to the `/wp-content/plugins/` directory
2. Activate Spinmyplanet from the 'Plugins' page in WordPress
3. Go to Images and add new images with minimum of 2 images to see the effect.(press ctrl key and then select images to select multiple images).
4. Copy the shortcode generated at images to any post.
5. Save the post and run on frontend.


= Usage for images spin =

Galleries are automatically appended to the bottom of your post/page unless you use the shortcode below. Using the shortcode will give you finer control over placement within the content area.


== Usage for woocomerce product ==
1. Go to Products and click Add product gallery images link on the right hand side and upload multiple images (minimum 2 or 3 images)
   (use ctrl key to select multiple images). 
2. Click on set product image and select an image similar to the image put in above step(select any 1 image from above multiple images to set your feature image). 
3. Save the product and run on frontend.


= Shortcode Usage =

Use the following shortcode anywhere in the content area to display the gallery

    [spin_my_image gallery_id="GALLERY_ID"]
